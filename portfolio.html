Easy Money

A savings tools for the not so financially savvy. I expanded off the given project brief to help the user achieve each goal. The user stories led me to creating low fidelity wireframes with simple buttons that required little financial understanding. Using user testing I received feedback + expanded on with the design process moving to the ui design + prototypes.

A responsive recipe + meal planning application with functioning mobile, tablet, and desktop screens. As the primary UX and UI role, I collected competitor analysis data, created user personas, ideation, user stories, wireframes, prototypes, and user testing.


Reverse Cotton

A branded retail boutqiue with native app designs for IOS + Android. Working through ideation, user stories, wireframes, prototypes, and user testing I created the boutique as the primary UI role.
